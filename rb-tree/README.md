# Red Black Tree

Prerequisite:
* C++23
* Graphvis

# Building
```
git clone https://github.com/ArsenySamoylov/graphs_course.git
mkdir build
cd build
cmake ../graphs_course/rb-tree/
make
```

Then trun tests (in build directory)
```
./tests/rb-tree-unit-tests.out
```

# Running
```
./rb-tree
```
Will run CLI

Best of luck =).
